Digital Humanities Projects

  • Voyant Visualization on Edgar Allan Poe stories

    • This project is its own webpage written similar to an essay format, with embedded text visualizations of 22 short stories written by Edgar Allan Poe, along with some analyses.

  • Twine Game

    • Twine is a website that allows users to create multi-linear works with control over content and style.

  • Omeka Site on 19th Century American Authors

    • This Omeka project is an archival website that contains four major exhibits on famous American male and female authors from the 19th Century.

  • Pedagogical Intervention

    • The pedagogical application is a lesson plan using concepts from digital humanities pedagogy.
